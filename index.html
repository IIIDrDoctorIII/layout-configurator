Layout Configurator - Comprehensive User Manual

About This Application

The Layout Configurator is a lightweight, interactive 2D layout planning tool. It is designed to help you easily draft and visualize floor plans, workshop arrangements, office spaces, and equipment configurations. With this tool, you can create designated zones, place solid objects, upload movable blueprint images, and annotate your plans with dynamic measurement lines. Your work can be saved locally and reloaded later for ongoing project management.

1. Global Controls (Top Bar)

  • Help: Opens this comprehensive user manual.
  • Undo: Reverts the canvas to the state before your last action.
  • Redo: Reapplies an action that was just undone.

2. Navigation & Viewport

  • Zoom In / Zoom Out: Click these buttons to adjust the scale of the entire workspace.
  • Desktop Panning: Hold the Alt key on your keyboard and click-drag anywhere on the canvas to pan (move your view) around the workspace. You can also use your mouse's scroll wheel to Zoom in and out dynamically.
  • Mobile Touch Navigation: Use a two-finger pinch gesture to Zoom in or out. Use a two-finger drag to Pan the workspace natively without interacting with objects.
  • 1-Finger Pan (Toggle): Tap this button to lock the canvas objects. While active, the button turns yellow, and you can drag across the screen with a single finger (or click-drag with a mouse) to pan your view. Tap again to return to selection and editing mode.

3. Adding & Managing Objects

  • + Rect, + Circle, + Triangle, + Right Tri: Click these buttons to drop new geometric shapes directly into the center of your current view.
  • + Measure Line: Drops a 1-dimensional measuring tool. This line has arrowheads on both ends and locks its vertical thickness. When stretched horizontally, the text label dynamically updates to show the real-time exact length.
  • Object List (Layers Menu): This section displays all items currently on your canvas, acting as a Z-index hierarchy.
    - Select: Click an item's text name in the list to select it on the canvas.
    - Lock (🔒 / 🔓): Click the padlock icon to lock an item. Locked items cannot be clicked, moved, rotated, or resized.
    - Reorder (▲ / ▼): Click the up or down arrows to move the item up or down in the layer stack.
    - Delete (X): Click the red X to permanently remove the item.
    - Drag-and-Drop (Desktop): Click and drag a row in the list to rapidly reorganize your layers.
  • Duplicate & Grouping: Select an object and click "Duplicate" to clone it quickly. You can also drag a selection box around multiple objects (or hold Shift while clicking) and use the "Group / Ungroup" button to bind them together.

4. View Toggles & Settings

  • Snap (ON/OFF): When enabled, moving and resizing objects will automatically snap them to the hidden 20-unit grid increments. Rotation will snap to precise 45-degree increments.
  • Grid (ON/OFF): Toggles the visual rendering of the background grid lines.
  • Dims (ON/OFF): Toggles the visibility of all on-canvas dimension overlays and custom name badges.
  • Text - / Text +: Decreases or increases the global font size for all measurement text and name badges.

5. Properties Panel

Select any unlocked object on the canvas to reveal its customizable properties.

  • Name: Type a custom label (e.g., "Main Workbench").
  • Color: Click the color swatch to open your device's native color picker.
  • Designation (Type): Changes the visual style of the shape (Zone, Object, Room, Measurement, or Blueprint).
  • Lock Aspect Ratio: When checked, typing a new Width will automatically calculate and update the Height to maintain proportions.
  • Dimensions (Width / Height): Type an exact number (in units/feet) to instantly resize an object.
  • Border Thickness: Adjust the stroke width for outlines and borders.
  • Rotation (Degrees): Type a specific angle, or click the +45° quick-turn button.

6. File & Export

  • Project Name: Type your project's title here. It will be used to automatically generate the filename.
  • Load Blueprint Image: Upload a `.jpg` or `.png` of a real floorplan.
  • Save / Load (JSON): Save your layout data to your local device, or load a previous file.
  • Clear All Canvas: Wipes all objects from your workspace. Click twice to confirm.

7. Keyboard Shortcuts

  • Delete / Backspace: Deletes the currently selected object(s).
  • Ctrl + Z: Undo the last action.
  • Ctrl + Y: Redo the last undone action.
  • Ctrl + D: Duplicate the currently selected object(s).
  • Shift + Click: Select multiple objects sequentially.
